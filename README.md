## 🤖 Agente Tutor Python

Este repositorio incluye un **agente personalizado** que funciona como tutor usando el **método socrático**:

### ¿Cómo usarlo?

1. Abre el chat de GitHub Copilot en VS Code
2. Menciona al agente: `@tutor-python`
3. Pídele que revise tus notas o te ayude a repasar

### Ejemplos de uso:

```
@tutor-python Revisa mis notas sobre decoradores y hazme preguntas

@tutor-python Ayúdame a entender mejor el concepto de regex

@tutor-python Quiero repasar lo que aprendí sobre manejo de archivos
```

### 🎓 Método Socrático

El agente **NO da respuestas directas**. En su lugar:
- ✅ Hace preguntas para guiar tu razonamiento
- ✅ Te ayuda a descubrir conceptos por ti mismo
- ✅ Profundiza en áreas que necesitas reforzar
- ❌ No crea contenido automáticamente
- ❌ No edita tus archivos

## 📖 Sobre el Curso

**Certificado Profesional de Google IT Automation con Python**

Este programa de 6 cursos enseña habilidades esenciales de IT automation:

1. 🐍 **Crash Course on Python** - Fundamentos del lenguaje
2. 💻 **Using Python to Interact with the Operating System** - Automatización de tareas
3. 🔀 **Introduction to Git and GitHub** - Control de versiones
4. 🐛 **Troubleshooting and Debugging Techniques** - Resolución de problemas
5. 🔧 **Configuration Management and the Cloud** - Gestión de configuración
6. 🤖 **Automating Real-World Tasks with Python** - Proyecto final
