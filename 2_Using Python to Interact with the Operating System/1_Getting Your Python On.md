En este módulo, aprendí la importancia de usar un IDE (Entorno de Desarrollo Integrado) para trabajar con Python. En mi caso, estoy utilizando Visual Studio Code, que es una herramienta muy versátil y eficiente.

# Fórmula de Automatización
Una fórmula útil para decidir si una tarea debe ser automatizada es:

**Tiempo que se tarda en automatizar < (Tiempo de la tarea * veces que se repite)**

Si el tiempo que se tarda en automatizar es menor, entonces vale la pena automatizar la tarea.

---

# Principio de Pareto
El principio de Pareto también se aplica en la administración de sistemas. Este principio establece que:

**El 20% de las actividades de un administrador de sistemas representa el 80% de los resultados.**

Por lo tanto, es importante identificar y priorizar esas actividades clave.

# Módulos útiles
He aprendido sobre dos módulos nuevos que son muy útiles para la automatización:

- **shutil**: Permite trabajar con archivos y directorios. Por ejemplo, se puede usar para verificar la cantidad de espacio libre en disco.
- **psutil**: Proporciona información sobre el uso de recursos del sistema, como el porcentaje de CPU en uso.

Estos módulos son herramientas poderosas para monitorear y gestionar recursos del sistema de manera eficiente.
